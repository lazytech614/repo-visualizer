# Contributing

Thank you for your interest in contributing.

## Development Setup

```bash
pnpm install
```

```bash
pnpm dev
```

---

## Branch Naming

Feature:

```text
feature/dependency-graph
```

Bugfix:

```text
fix/cycle-detection
```

---

## Commit Format

```text
feat: add complexity analysis

fix: resolve dependency graph bug

docs: update README
```

---

## Pull Requests

Include:

- Description
- Screenshots
- Testing Notes
