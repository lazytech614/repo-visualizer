# Analyzer Engine

The analyzer package is responsible for repository inspection.

## Components

### Tree Analyzer

Builds repository hierarchy.

### Dependency Analyzer

Parses import statements.

### Cycle Detector

Detects circular dependencies.

### Dead File Detector

Finds unused files.

### Complexity Analyzer

Computes complexity metrics.

### Health Score Calculator

Generates repository health score.

### Summary Generator

Creates AI context.
